# 🚀 Alien Invasion

Космическая аркада на Python + Pygame.

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| ← → | Движение корабля |
|Space| Стрельба |
| Q | Выход |

## 📦 Установка

```bash
# Требуется Python 3.11.x
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python alien_invasion.py
